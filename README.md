# Sales Analysis SQL Project
**By Vatsal Barot**

## 📊 Project Overview
Analysis of online store sales data using MySQL
to uncover business insights and recommendations.

## 🛠️ Tools Used
- MySQL
- SQL (SELECT, WHERE, GROUP BY, JOIN, UPDATE)

## 📁 Files
- `sales_analysis.sql` — All queries with data

## 🔍 Key Insights
1. Ayesha is the highest spender at Rs.3,500
2. March and April have equal revenue at Rs.4,000
3. Shoes is the most expensive product
4. T-shirt is the cheapest product at Rs.500

## 💡 Recommendation
Launch a Valentine's Day campaign in February
to boost sales during the slowest month of the year.
